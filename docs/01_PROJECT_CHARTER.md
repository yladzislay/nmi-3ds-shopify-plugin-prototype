# Устав Проекта: NMI Shopify Integration

## 1. Контекст
Проект инициирован для разработки кастомного платежного приложения (Custom Payment App) для платформы Shopify. Цель — обеспечить прием платежей через шлюз NMI с обязательной поддержкой протокола безопасности 3D Secure (3DS).

## 2. Бизнес-цели
*   **Безопасность:** Обеспечить соответствие требованиям PSD2 и PCI DSS.
*   **Конверсия:** Создать бесшовный UX на чекауте, минимизируя уход пользователей во время оплаты.
*   **Универсальность:** Решение должно быть готово к масштабированию и внедрению в реальные магазины.

## 3. Роли и Обязанности
*   **Shopify Developer (User/You):**
    *   Разработка Frontend части (Checkout Extensions).
    *   Разработка Backend части (Middleware).
    *   Интеграция NMI SDK/API.
*   **AI Assistant (Gemini/Me):**
    *   Бизнес-анализ и декомпозиция задач.
    *   Генерация кода и архитектурных решений.
    *   Code Review и оптимизация.

## 4. Критерии Успеха
1.  Успешное прохождение тестового сценария оплаты (Happy Path).
2.  Корректная обработка сценария 3D Secure (Redirect Flow).
3.  Отсутствие критических ошибок при сетевых задержках или отказах банка.
4.  Чистый, поддерживаемый код с документацией.
